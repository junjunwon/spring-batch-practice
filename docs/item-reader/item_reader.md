# 데이터베이스 입력
- JDBC, 하이버네이트, JPA 등을 통해 데이터베이스에서 데이터를 읽어올 수 있도록 스프링 배치가 기본적으로 제공하는 기능

## JDBC
스프링이 기본으로 제공하는 JDBCTemplate을 사용했을때
    JdbcTemplate이 전체 ResultSet에서 한 row씩 순서대로 가져오면서, 모든 로우를 필요한 도메인 객체로 변환해 메모리에 저장하는 문제
스프링 배치는 이를 해결하기 위해 한 번에 처리할 만큼의 레코드만 메모리에 로딩하는 cursor와 paging 기법을 제공한다.
